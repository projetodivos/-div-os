Do zero ao código: HTML

🖥️ HTML Básico: Guia Nerd para Iniciantes

Se você está entrando no mundo do desenvolvimento web, o HTML é o ponto de partida. Ele é como a “espinha dorsal” da internet — dá a estrutura, define o que aparece na tela e organiza o conteúdo.

O segredo está nas tags, que são como comandos que dizem ao navegador: “ei, isso aqui é um título!” ou “isso aqui é uma imagem!”. Vamos conhecer as principais:


🔹 Introdução ao HTML

O HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas da web. Todo documento HTML começa com uma estrutura básica que informa ao navegador como interpretar o conteúdo.

📌 Estrutura inicial de um documento HTML

<!DOCTYPE html>  
<html lang="pt-BR">  
  <head>  
    <meta charset="UTF-8">  
    <meta name="viewport" content="width=device-width, initial-scale=1.0">  
    <title>Minha Primeira Página</title>  
  </head>  
  <body>  
    <h1>Olá, Mundo!</h1>  
    <p>Este é o meu primeiro parágrafo em HTML.</p>  
  </body>  
</html>

🔎 Explicando os principais elementos

  • <!DOCTYPE html>: informa ao navegador que o documento está usando HTML5.
  • <html lang="pt-BR">: define o idioma da página.
  • <meta charset="UTF-8">: garante que caracteres especiais como acentos sejam exibidos corretamente.
  • <meta name="viewport">: ajuda a página a se adaptar a diferentes tamanhos de tela (responsividade).
  • <title>: define o título que aparece na aba do navegador.
  • <h1> até <h6>: títulos e subtítulos, do mais importante (<h1>) ao menos relevante (<h6>).
  • <p>: representa um parágrafo de texto.

🔹 Títulos: <h1> até <h6>

Essas tags definem cabeçalhos.

  • <h1> é o mais importante (usado geralmente no título principal da página).
  • <h2> até <h6> são subtítulos, cada vez menores em importância e tamanho.

<h1>Meu Site</h1>
          <h2>Seção de Notícias</h2>

🔹 Parágrafos: <p>

Tudo o que você escreve como texto corrido vai dentro de <p>. É a forma padrão de criar blocos de texto.

👉 Exemplo:

(imagem vscode)

<p>Esse é um parágrafo de exemplo.</p>

🔹 Links: <a>

Com a tag <a>, você cria links para outros sites ou páginas.

  • O atributo href indica o destino.
  • O atributo target=”_blank” abre em uma nova aba.

👉 Exemplo:

(imagem vscode)

<a href="https://www.wikipedia.org" target="_blank">Ir para a Wikipédia</a>

🔹 Imagens: <img>

Usamos <img> para exibir imagens.

  • O atributo src é o caminho da imagem.
  • O atributo alt descreve a imagem (importante para acessibilidade e SEO).

👉 Exemplo:

(imagem vscode)

<img src="imagem.jpg" alt="Descrição da imagem">

🔹 Listas: <ul>, <ol> e <li>

  • <ul> → lista não ordenada (bolinhas).
  • <ol> → lista ordenada (números).
  • <li> → cada item da lista.

🔹 Quebra de linha: <br>

Funciona como um "Enter". Use quando quiser forçar uma quebra de linha sem abrir um novo parágrafo.

👉 Exemplo:

<p>Linha 1<br>Linha 2</p>
