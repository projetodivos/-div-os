            <h1>🖥️ HTML Básico: Guia Nerd para Iniciantes</h1>
          
            <p>Se você está entrando no mundo do desenvolvimento web, o <strong>HTML</strong> é o ponto de partida. Ele é como a “espinha dorsal” da internet — dá a estrutura, define o que aparece na tela e organiza o conteúdo.</p>
          
            <p>O segredo está nas <strong>tags</strong>, que são como comandos que dizem ao navegador: <em>“ei, isso aqui é um título!”</em> ou <em>“isso aqui é uma imagem!”</em>. Vamos conhecer as principais:</p>
          
            <hr>
          
            <h2>🔹 Títulos: &lt;h1&gt; até &lt;h6&gt;</h2>
          
            <p>Essas tags definem cabeçalhos.</p>
          
            <ul>
              <li>&lt;h1&gt; é o mais importante (usado geralmente no título principal da página).</li>
              <li>&lt;h2&gt; até &lt;h6&gt; são subtítulos, cada vez menores em importância e tamanho.</li>
            </ul>
          
            <p>(imagem vscode)</p>
          
            <pre><code>&lt;h1&gt;Meu Site&lt;/h1&gt;
          &lt;h2&gt;Seção de Notícias&lt;/h2&gt;
          &lt;h3&gt;Notícia 1&lt;/h3&gt;</code></pre>
          
            <h2>🔹 Parágrafos: &lt;p&gt;</h2>
          
            <p>Tudo o que você escreve como texto corrido vai dentro de &lt;p&gt;. É a forma padrão de criar blocos de texto.</p>
          
            <p>👉 Exemplo:</p>
          
            <p>(imagem vscode)</p>
          
            <pre><code>&lt;p&gt;Esse é um parágrafo de exemplo.&lt;/p&gt;</code></pre>
          
            <h2>🔹 Links: &lt;a&gt;</h2>
          
            <p>Com a tag &lt;a&gt;, você cria links para outros sites ou páginas.</p>
          
            <ul>
              <li>O atributo href indica o destino.</li>
              <li>O atributo target=”_blank” abre em uma nova aba.</li>
            </ul>
          
            <p>👉 Exemplo:</p>
          
            <p>(imagem vscode)</p>
          
            <pre><code>&lt;a href="https://www.wikipedia.org" target="_blank"&gt;Ir para a Wikipédia&lt;/a&gt;</code></pre>
          
            <h2>🔹 Imagens: &lt;img&gt;</h2>
          
            <p>Usamos &lt;img&gt; para exibir imagens.</p>
          
            <ul>
              <li>O atributo src é o caminho da imagem.</li>
              <li>O atributo alt descreve a imagem (importante para acessibilidade e SEO).</li>
            </ul>
          
            <p>👉 Exemplo:</p>
          
            <p>(imagem vscode)</p>
          
            <pre><code>&lt;img src="imagem.jpg" alt="Descrição da imagem"&gt;</code></pre>
          
            <h2>🔹 Listas: &lt;ul&gt;, &lt;ol&gt; e &lt;li&gt;</h2>
          
            <ul>
              <li>&lt;ul&gt; → lista não ordenada (bolinhas).</li>
              <li>&lt;ol&gt; → lista ordenada (números).</li>
              <li>&lt;li&gt; → cada item da lista.</li>
            </ul>
          
            <ul>
              <li>HTML</li>
              <li>CSS</li>
              <li>JavaScript</li>
            </ul>
          
            <ol>
              <li>Estudar HTML</li>
              <li>Praticar exercícios</li>
              <li>Criar projetos</li>
            </ol>
          
            <h2>🔹 Quebra de linha: &lt;br&gt;</h2>
          
            <p>Funciona como um "Enter". Use quando quiser forçar uma quebra de linha sem abrir um novo parágrafo.</p>
          
            <p>👉 Exemplo:</p>
          
            <pre><code>&lt;p&gt;Linha 1&lt;br&gt;Linha 2&lt;/p&gt;</code></pre>
