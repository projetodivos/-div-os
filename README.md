# Projeto <div>os

## Visão Geral

O **Projeto `<div>os`** é uma página web desenvolvida em **HTML, CSS e Java**, voltada para quem deseja aprender HTML de forma gratuita e acessível.

O conteúdo atende tanto iniciantes que nunca tiveram contato com a área quanto pessoas que já atuam em tecnologia e querem revisar fundamentos.

Nosso objetivo é criar uma página **minimalista, funcional e clara**, com explicações simples, exemplos visuais e exercícios práticos que aproximem a teoria de situações reais.
